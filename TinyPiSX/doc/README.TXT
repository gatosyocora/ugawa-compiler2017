[発展演習14  比較演算子 ==, !=, >, <, >=, <= の実装]

if文やwhile文の処理を制御する際の条件に用いる演算子である.
これらの演算子の左右に書かれた値を評価し, 
その後, 左右の値の評価結果に対して演算子に対応した比較を行う.

インタプリタの場合, 
比較において左右の値の評価結果の間に演算子にあった関係が
  満たされていれば, trueを意味する1を
  満たされていなければ, falseを意味する0を
関数の戻り値として返す.

コンパイラの場合, 
比較において左右の値の評価結果の間に演算子にあった関係が
  満たされていれば, trueを意味する1を
  満たされていなければ, falseを意味する0を
r0に代入し, 次の処理へ移る.

以下に各比較演算子が左右の値の評価結果に対してどのような比較を行うか示す.
  == : 左右の値が同じであるか
  != : 左右の値が異なっているか
  >  : 左の値が右の値より大きいか
  <  : 左の値が右の値より小さいか
  >= : 左の値が右の値以上であるか
  <= : 左の値が右の値以下であるか
これらは正負の値の評価結果にも対応している.
  
各比較演算子の結合の方向は左から右となっている.
また, 以下に示す実装済みの演算子らにおいて上のほうが優先順位が高くなっている.
  ()			括弧
  - ~			単項演算子
  * /			乗除演算子
  + -			加減演算子
  < <= > >=		比較演算子
  == !=			等値演算論理演算子子
  &				ビットAND演算子
  |				ビットOR演算子
  =				代入演算子
  
テストプログラム
  TinyPiSX/test/141.pisx     ifelseと==を用いたプログラム
  TinyPiSX/test/142.pisx     ifelseと!=を用いたプログラム
  TinyPiSX/test/143.pisx     ifelseと>を用いたプログラム
  TinyPiSX/test/144.pisx     ifelseと<を用いたプログラム
  TinyPiSX/test/145.pisx     ifelseと>=を用いたプログラム
  TinyPiSX/test/146.pisx     ifelseと<=を用いたプログラム
  TinyPiSX/test/147.pisx     whileと>=を用いたプログラム
  TinyPiSX/test/148.pisx     比較演算子の優先順位を確かめるプログラム

[発展演習15  論理演算子 &&, ||, ! の実装]

if文やwhile文の処理を制御する際の条件に用いる演算子である。
&&, || はこれらの演算子の左右に書かれた式を評価し, 
その後, 左右の式の評価結果に対して演算子に対応した比較を行う.
! はこの演算子の右に書かれた式を評価し, その後, 右の式の評価結果に反する結果を返す.
左右に書かれたものが式ではなく値であるとき, 値が1以上であるならその評価結果は1として扱う.

&&, || において
  インタプリタの場合, 
  比較において左右の式の評価結果の間に演算子にあった関係が
    満たされていれば, trueを意味する1を
    満たされていなければ, falseを意味する0を
  関数の戻り値として返す.
  
  コンパイラの場合, 
  比較において左右の式の評価結果の間に演算子にあった関係が
    満たされていれば, trueを意味する1を
    満たされていなければ, falseを意味する0を
  r0に代入し, 次の処理へ移る.
  
  以下に各比較演算子が左右の式の評価結果に対してどのような比較を行うか示す.
    && : 左右の式の評価結果がどちらも1(true)であるか
    || : 左右の式の評価結果のいずれかが1(true)であるか
    
! において
  インタプリタの場合, 
  演算子の右に書かれた式の評価結果が
    1(true)であれば, falseを意味する0を
    0(false)であれば, trueを意味する1を
  関数の戻り値として返す.
  
  コンパイラの場合, 
  演算子の右に書かれた式の評価結果が
    1(true)であれば, falseを意味する0を
    0(false)であれば, trueを意味する1を
  r0に代入し, 次の処理へ移る.
  
  
各比較演算子の結合の方向は左から右となっている.
また, 以下に示す実装済みの演算子らにおいて上のほうが優先順位が高くなっている.
  ()			括弧
  - ~ !			単項演算子
  * /			乗除演算子
  + -			加減演算子
  < <= > >=		比較演算子
  == !=			等値演算子
  &				ビットAND演算子
  |				ビットOR演算子
  &&			論理AND演算子
  ||			論理OR演算子
  =				代入演算子
  
テストプログラム
  TinyPiSX/test/151.pisx    &&を用いたプログラム
  TinyPiSX/test/152.pisx    ||を用いたプログラム
  TinyPiSX/test/153.pisx    !を用いたプログラム
  TinyPiSX/test/153.pisx    &&, ||, !を用いたプログラム
